Universal real-time state synchronization for Python.
Manipulate memory as if the network didn't exist.

pip install easysync
server.py ● Master Node
from easysync import SyncedObject, SyncServer

server = SyncServer(port=5000)
server.start_thread()

@SyncedObject(server)
class State:
    points = 0
    users = []

state = State()
# Variables below are network-bound

state.points = 0
state.users = []

Why EasySync?

Universal Transparent Proxy

Forget about sockets. EasySync wraps your variables without modifying application logic through pure Python metadata handling (__getattribute__).

Binary Performance

No JSON text overhead. EasySync serializes via Pickle and frames data in raw bytes for extremely fast network synchronization.

Built for Data Science

Optimized handling of C-level memory for direct copy-and-reassign of heavy NumPy, Pandas and Scikit-Learn structures.

Native Asyncio Server

The server is built on asyncio.start_server to handle thousands of simultaneous connections without CPU overhead or blocking threads.

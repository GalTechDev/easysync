EasySync

Synchronisation universelle d'états asynchrones pour Python.
Manipulez la mémoire comme si le réseau n'existait pas.

pip install easysync
serveur.py ● Master Node
from easysync import SyncedObject, SyncServer

server = SyncServer(port=5000)
server.start_thread()

@SyncedObject(server)
class State:
    points = 0
    users = []

state = State()
# Les variables ci-dessous sont liées au réseau

state.points = 0
state.users = []

Pourquoi EasySync ?

Proxy Universel Translucide

Oubliez les sockets. EasySync enveloppe vos variables sans modifier la logique applicative via une gestion pure des métadonnées Python (_getattribute_).

Performances Binaires (Latence < 20ms)

Aucun overhead JSON textuel. EasySync sérialise via Pickle et structure ses trames en octets purs pour garantir une synchronisation réseau extrêmement rapide.

Taillé pour la Data Science

Gestion optimisée des mémoires C Pures pour autoriser la copie et la ré-assignation directe de structures lourdes de Numpy Pandas et Scikit-Learn.

Serveur Asyncio Natif

Le serveur repose sur asyncio.start_server pour gérer des milliers de connexions simultanées sans surcharge CPU ni threading bloquant.
